쿠팡 광고 계산기 HTML 앱을 열어주는 스킬이야.

> **🚀 처음 사용하는 환경이라면**: 같은 폴더의 `heyclaude.md` 참고. macOS면 별도 세팅 없이 바로 동작.

아래 명령어를 실행해서 HTML 파일을 브라우저로 열어줘:

```bash
open ~/.claude/skills/ads-manager-app/scripts/ad-calculator.html
```

실행 후 "쿠팡 광고 계산기가 열렸어요." 라고 알려줘.
