## 1. 목표 ROAS 설정

### 계산식
```
광고수익률(ROAS) = 광고 전환 매출 ÷ 집행 광고비
예) 광고비 100만원 → 광고매출 500만원 = ROAS 500%
```

### 핵심 원칙
- **목표 ROAS는 보장값이 아님** — 시스템이 입찰가를 결정하는 참고 입력값
- **너무 높게 설정 = 오히려 노출 감소** — 높은 목표 달성을 위해 알고리즘이 입찰가를 자동으로 낮춤
- **제안 광고수익률 기준으로 시작** 권장 (광고센터에서 확인 가능)
- 처음엔 보수적으로 → 최적화 후 점진적으로 높이기

### 학습/최적화 기간
- **7~14일 소요** (매출 최적화 광고, AI 스마트 광고)
- **이 기간 중 목표 ROAS나 예산 변경 금지** — 변경 시 학습 리셋, 기간 추가 소요
- 실제 ROAS가 목표 초과해도 광고는 중단되지 않음

### ⚠️ 현재 상황 해석
> 목표 1,000% (제안값 280%)는 3.5배 높은 설정 → 시스템이 입찰가를 낮춰 노출 억제 상태.
> 단, 실제 ROAS 5,660%는 전환이 일어나고 있다는 신호.
> 1500% → 1000%로 낮춘 것은 올바른 방향. 제안값(280%)까지 단계적으로 낮추는 것 고려.

## 2. 일예산 운영

### 기준
- 최소: 10,000원/일
- **권장: 30,000원 이상** (피크타임 노출 기회 확보)
- 추천 일예산: 광고센터 > 캠페인 목록 > 예산 탭에서 확인 가능

### 탄력 예산 운영
- 미사용 금액은 동일 월 내 예산 부족한 날에 이월 사용
- **최대 일예산의 20%까지 추가 사용 가능**
- 예산이 소진되면 자동 광고 중단 → 다음날 0시 재시작

### 예산이 소진되지 않는 이유 (진단 체크리스트)
1. 목표 ROAS가 너무 높게 설정됨 → 입찰가 낮아짐
2. 등록 상품 수 부족
3. 경쟁 입찰가 대비 낮은 입찰가
4. 아이템위너 미선정 또는 품절 상품 포함

### 주의
- **매일 ON/OFF 금지** — 최적화 방해. 자동규칙 사용 권장
- `골든타임(저녁 피크) 예산 부족 방지` 자동규칙 설정 권장

## 3. 캠페인 & 상품 구조

### 상품 등록 원칙
- **하나의 상품은 하나의 캠페인에만 등록** (중복 등록 시 비효율)
- 캠페인당 최대 500개 상품 등록 가능
- 동일 카테고리 옵션들은 묶어서 운영 가능
- 완전히 다른 카테고리 상품은 분리 운영

### 광고 유형 4종
| 유형 | 특징 | 적합 상황 |
|------|------|-----------|
| AI 스마트 광고 | 키워드·입찰가·상품 전부 자동 | 초보, 운영 자동화 |
| 매출 최적화 광고 | 상품 선택 후 ROAS+예산만 설정 | 성과 중심 운영 |
| 수동 성과형 광고 | 상품·키워드·입찰가 수동 설정 | 전략적 세밀한 운영 |
| 매출 스타트 광고 | 노출 부족 신상품 집중 노출 | 신상품 첫 판매 확보 |

### 성공 권장사항 (매출 최적화 광고 기준)
1. 권장 일예산 이상 설정 (최소 3만원)
2. 상품효율지수 기준 충족
3. 제안 광고수익률로 설정
4. 중복 상품 제거

## 4. 성과 분석

### 전환 집계 기준
- **광고 클릭 후 14일 이내** 구매 = 전환으로 집계
- 직접 전환: 광고 클릭 상품 = 구매 상품
- 간접 전환: 광고 클릭 후 동일 셀러의 다른 상품 구매

### 보고서 단위
- 캠페인 / 광고그룹 / **상품별** / **키워드별** / 일별 — 모두 제공
- 광고센터 > 광고보고서 > 엑셀 다운로드 가능
- 자동 광고 보고서: 정기 리포트 자동 수신 설정 가능

### ROAS 낮을 때 체크리스트
1. 상품 상세페이지 업데이트
2. 리뷰 쌓기 (상품 경쟁력 확보)
3. 가격 경쟁력 점검

## 5. 현재 캠페인 진단 (캐리어바퀴_홍성_260415)

| 항목 | 현재값 | 공식 기준 | 판단 |
|------|--------|-----------|------|
| 일예산 | 100,000원 | 권장 3만원 이상 | ✅ 충분 |
| 목표 ROAS | 1,000% | 제안값 280% | ⚠️ 높음 (노출 억제) |
| 실제 ROAS | 5,660% | — | 🔴 목표보다 훨씬 높음 |
| 클릭수 | 5~7회/일 | — | ⚠️ 매우 적음 |
| 전환율 | 20~50% | 만원대 15~20% | ✅ 우수 |
| 학습기간 중 변경 | ROAS 1500→1000% | 변경 금지 | ⚠️ 재학습 발생 |

### 시사점
- 목표 ROAS가 높아서 AI가 입찰가를 낮게 잡고 있음 → 클릭이 적은 근본 원인
- 전환율 자체는 매우 우수 → 상품 경쟁력 있음
- 102번 외발휠에 광고 집중 → 나머지 옵션 기회 부족 (정상적인 AI 동작)
- ROAS를 제안값(280%) 방향으로 단계적 인하 고려

## 6. 세팅 단계별 공식 가이드 (CDP 직접 수집)

### 매출 최적화 광고 세팅 순서 (settings/mps)
1. 캠페인 만들기 → 광고 목표: **매출 성장** 선택
2. 기간: **종료일 없음** 권장 (특정 기간 설정 시 만료 후 재시작하면 재학습 발생)
3. 상품 선택: **수동 상품 설정** → 최대 500개
   - **광고효율UP 태그** 상품 우선 등록 권장
   - 다른 캠페인과 **중복 상품 제거** 필수
4. 운영 방식: **자동운영** 선택 (CPC + 탄력일예산)
5. 일예산: 추천 일예산 참고, 최소 3만원 이상
   - 미소진 예산 → 같은 달 예산 부족한 날 자동 이월 사용
6. 목표 ROAS: 제안값 기준으로 설정

### AI 스마트 광고 특이사항 (settings/aps)
- **계정당 1개만** 생성 가능
- 전체 상품 자동 일괄 등록 (실시간 수요 기반 자동 선택)
- 매출 최적화 캠페인에 이미 등록된 상품은 **자동 제외**됨
- 특정 상품 제외 가능 (개별 제외 / 대량 제외)
- 두 캠페인 동시 운영 시: 매출최적화 우선, AI스마트는 나머지 상품 담당

### CPC 과금 방식 핵심
- **차순위 입찰 방식**: 내가 이긴 경쟁자의 입찰가 + 1원만 청구
- 광고 노출 영역: 검색결과 상단/중단, 카테고리 페이지, 상품 상세페이지, 장바구니, 메인페이지

## 7. 광고할 상품 선택 기준

### 광고효율UP 배지 (blog/73)
- 쿠팡 AI가 클릭률·전환율 예측 기반으로 **효율 높을 상품에만** 자동 부여
- 배지 없는 상품보다 **광고효율UP 배지 상품 우선 광고** 권장
- AI 스마트 광고 사용 시: 알고리즘이 실시간 수요 기반으로 상품 자동 선택 → 직접 선택 불필요

### 성공 권장사항 4가지 (blog/50)
- 광고관리 > 캠페인 대시보드 > **[성공권장사항] 컬럼**에서 미충족 항목 자동 알림
- 4가지 모두 충족 시 노출 효율 + 목표 ROAS 달성 확률 상승 (쿠팡 자체 분석 기준)
- 수정 후 반영까지 최대 2시간 소요
- 주의: 충족이 ROAS를 보장하지는 않음

### AI 기반 상품명 자동 최적화 (blog/95)
- 캐러셀 광고 지면(메인홈, 검색결과, 상세페이지)에서 AI LLM이 10자/20자 제한에 맞게 상품명 자동 재구성
- 실전 팁: **상품명에 브랜드·제품유형·핵심 키워드** 미리 넣어두면 AI 최적화 효과 극대화
- 확인: 광고센터 > 상품 기회 탐색 > "광고 영역 상품명" 컬럼

## 8. 성과 분석 핵심 용어 (analysis/01)

| 용어 | 설명 |
|------|------|
| 노출수 | 광고가 고객에게 표시된 횟수 |
| 클릭률(CTR) | 클릭수 ÷ 노출수 × 100. 썸네일·가격·리뷰 매력도 판단 지표 |
| 광고 전환 판매수 | 클릭 후 14일 내 구매 건수 (직접+간접 전환 합계) |
| 광고 전환 주문수 | 클릭 후 14일 내 주문 건수 (3개 동시 구매 = 판매수 3, 주문수 1) |
| 오늘 누적 광고비 | 당일 현재까지 사용된 광고비 (최대 10분 지연) |

> 전환 = 직접 전환 + 간접 전환 (광고 클릭 후 내 **다른 상품** 구매도 포함)

### 보고서 분석 실전 팁 (analysis/03)
- 기간=지난주, 기간구분=일별, 모든 캠페인 선택 → 일별 성과 트렌드 파악
- 엑셀 다운로드 → "총 판매수량(14일)" 컬럼에서 **0 제외 필터링** → 실제 전환 발생 상품만 추려서 분석
- 요약보기 탭: 총 전환매출액 / 광고비 / 노출수 / 총 ROAS 4대 지표 카드 확인

## 9. 광고비 소진 안 될 때 점검 체크리스트

### 5가지 점검 항목 (guide/07)
1. **목표 ROAS 과다 설정** → 추천값으로 낮추기
2. **광고 상품 수 + 상품 효율지수** 기준 이상 충족 여부
3. **제외 키워드 과다** → 매출최적화/AI스마트는 키워드 자동 매칭, 제외가 오히려 노출 감소
4. **캠페인 상태 '학습 부족'** → 성공 권장사항 충족 + 상품 경쟁력 강화 필요
5. **상품 정보 매력도** → 썸네일(전체샷), 상품명(정확한 명칭) 개선 → 클릭률 상승

### 상품 효율지수
> "선택한 상품의 과거 구매 데이터를 기반으로 충분한 상품이 등록되었는지 보여주는 기능"
- 최소 기준 이상, 권장 기준까지 채우면 효과적인 캠페인 운영
- 캠페인 등록 단계에서 확인 가능

### 구매 전환 상세페이지 체크리스트 (blog/66)
- 화려한 디자인보다 **정확한 이미지 + 핵심 텍스트 배열**
- 긴 설명 대신 핵심 요약
- 시각적으로 단순한 구성
- 광고센터 > 상세페이지 관리에서 무료 편집기 제공 (로켓그로스/마켓플레이스만)

### 캠페인 학습/최적화 기간 (videoclass/29)
- 성장미션 + 성공권장사항 달성 후 학습 완료 → 효율 안정화
- 매출최적화/AI스마트/매출스타트는 키워드 자동 매칭 (직접 수정 불필요)
- 수동 성과형만 키워드 직접 설정 가능
